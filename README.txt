# Cómo actualizar tus archivos (pasos simples)
1) Entra a tu repositorio `anelariel.github.io` en GitHub.
2) Haz clic en **Add file → Upload files**.
3) Sube los archivos de esta carpeta:
   - `index.html` (la página principal)
   - `favicon.svg` (icono de la pestaña)
   - `robots.txt` (permite a Google leer tu sitio)
   - `sitemap.xml` (ayuda a que aparezca en Google)
4) Pulsa **Commit changes** (guardar) y espera 1 minuto.
5) Abre `https://anelariel.github.io` para ver los cambios.
